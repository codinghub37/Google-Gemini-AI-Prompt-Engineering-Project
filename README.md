# Gemini Prompt Engineering Project

This project demonstrates various prompt engineering techniques using Google's Gemini API.

## Tasks Included

- AI Question Answering
- Conversational Prompting
- Travel Planning Assistant
- Sentiment Analysis
- Mathematical Reasoning
- Resume Information Extraction

## Technologies

- Python
- Google Gemini API
- Prompt Engineering

## Examples

1. General Question Answering
2. Travel Itinerary Generation
3. Sentiment Classification
4. Step-by-Step Reasoning
5. Structured JSON Extraction
